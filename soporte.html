Preguntas Frecuentes

Las respuestas más comunes sobre el proceso de inscripciones tardías

¿Qué son las inscripciones tardías?

Las inscripciones tardías son un periodo especial que permite a los estudiantes inscribir materias después del cierre del periodo ordinario de inscripciones. Este proceso está sujeto a disponibilidad y aprobación del departamento correspondiente.

¿Cuándo están disponibles las inscripciones tardías?

Las inscripciones tardías estarán disponibles cuando sean habilitadas por la administración de cada departamento. Te recomendamos estar atento a los avisos oficiales y consultar con tu departamento.

¿Qué requisitos necesito para inscribirme?

Los requisitos básicos son: cédula de identidad vigente, correo institucional activo (@unet.edu.ve), teléfono de contacto y conocer tu semestre actual. Algunos departamentos pueden tener requisitos adicionales.

¿Cómo verifico mi identidad en el sistema?

El sistema envía un código de verificación a tu correo institucional. Debes ingresar este código en la página de verificación para confirmar tu identidad y continuar con el proceso.

¿Puedo inscribir cualquier materia?

Puedes inscribir solo las materias y secciones disponibles en el sistema. Algunas materias pueden tener requisitos previos o cupos limitados. Verifica la disponibilidad antes de inscribirte.

¿Qué hago si no recibo el código de verificación?

Si no recibes el código en 5 minutos, verifica tu carpeta de spam. Si aún no lo recibes, puedes reenviar el código desde la página de verificación. Si persiste el problema, contacta a soporte técnico.

¿Puedo modificar mi inscripción después de confirmarla?

Una vez confirmada la inscripción, no se pueden realizar modificaciones a través del sistema. Si necesitas hacer cambios, debes contactar directamente con tu departamento académico.

¿El sistema está disponible para todos los departamentos?

Actualmente el sistema está disponible solo para el Departamento de Matemática. Los demás departamentos se irán incorporando gradualmente. Consulta la página de departamentos para ver la disponibilidad.
